Example Notebooks
=================

The following notebooks demonstrate scHopfield's capabilities on a hematopoiesis
dataset from Paul et al. (2015).

To run locally, convert the ``.py`` files in ``notebooks/`` to Jupyter notebooks
using Jupytext or ``jupyter nbconvert``, then place the resulting ``.ipynb``
files in ``docs/notebooks/``.
