Network Analysis
================

.. note::
   Convert ``notebooks/03_network_analysis.py`` to
   ``docs/notebooks/03_network_analysis.ipynb`` using Jupytext or
   ``jupyter nbconvert``, then this page will render the full notebook.

This notebook covers:

- Network correlation dendrograms based on RV coefficient and Pearson similarity
- Symmetricity analysis of W matrices (symmetric ≈ equilibrium,
  skew-symmetric ≈ oscillatory)
- Network centrality metrics (degree, betweenness, eigenvector) and top-gene tables
- Eigenanalysis of interaction matrices: spectra, eigenvector gene loadings,
  and extreme eigenvalue tables
- GRN visualisation: network graphs and interaction heatmaps
