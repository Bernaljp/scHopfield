Perturbation Analysis
=====================

.. note::
   Convert ``notebooks/05_perturbation_analysis.py`` to
   ``docs/notebooks/05_perturbation_analysis.ipynb`` using Jupytext or
   ``jupyter nbconvert``, then this page will render the full notebook.

This notebook covers:

- ODE-based single-cell trajectory simulation
- ODE-based gene perturbation (knockout / overexpression) for single cells
- Population-level cell fate shift using ODE dynamics and Hopfield flow
- GRN propagation-based perturbations (CellOracle-compatible approach)
- Perturbation effect heatmaps and top-affected-gene bar charts
- Side-by-side comparison of multiple perturbation conditions
